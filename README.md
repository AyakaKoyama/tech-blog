## 開発の背景

Next.jsのキャッチアップに加えて、
<br />
この技術ブログの開発を通じて**社内メンバー間でのナレッジシェアを活性化させるという想定で設計**しました。
<br />
実際の運用は行っていませんが、本アプリの開発を通じてNext.jsを用いたデータの取得表示・microCMSを用いた情報の管理等を新しく学ぶことができました。
